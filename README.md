\# Wumpus World Logic Agent



This project implements a knowledge-based AI agent that navigates a Wumpus world using propositional logic and resolution.



\## Features

\- Dynamic grid

\- Logical inference

\- Web visualization



\## How to Run

1\. Run backend:

&#x20;  python backend/agent.py



2\. Open frontend:

&#x20;  frontend/index.html
